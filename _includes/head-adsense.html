{% comment %} head-adsense.html ----------------- Google AdSense — global publisher script. Loaded only when google_adsense.enabled is true in _data/integrations.yml. auto_ads: true → enables AdSense Auto Ads site-wide. AdSense will automatically find and fill ad slots across all pages. auto_ads: false → loads the AdSense script for manual ad units only. Place elements where you want ads to appear. NOTE: The AdSense script uses crossorigin="anonymous" and async to avoid render-blocking. Do not add defer — AdSense requires async specifically.
